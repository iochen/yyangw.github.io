相较于 Windows 版本,Linux 版本的缺陷

  • 无法显示菜单,需要使用 CTRL-ALT-PGDN 打开
  • 鼠标不能很好的工作
  • 无法实现全屏
  • 无法运行视频加速,所以性能不如 Windows 版本
